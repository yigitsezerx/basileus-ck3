# =====================================================================
# BASILEUS — REORGANIZE THE THEMES
# ---------------------------------------------------------------------
# A multi-option (widget) decision: the e_byzantium holder, having fully
# secured a frontier region, consolidates it into a historical theme —
# merging the region's duchies into one surviving duchy, renaming it, and
# destroying the emptied parent duchies. One-time per theme (locked via the
# unavailable_unique_decisions global list). Uses CK3's vanilla decision
# option-list framework (controller = decision_option_list_controller).
#
# Adding a theme = one item below + one basileus_form_theme_X_effect +
# one lock flag + its loc keys.
# =====================================================================
